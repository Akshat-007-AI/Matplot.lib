# Data Visualization using Matplotlib

## Project Overview
This project demonstrates how to create different types of data visualizations using the Matplotlib library in Python. The main objective of this project is to understand how graphical representations help in analyzing and interpreting data effectively. Using Matplotlib, several types of charts such as line plots, bar charts, and pie charts are created to present data in a clear and visually appealing way.

## Features
- Line Plot Visualization
- Bar Chart Representation
- Pie Chart Visualization
- Simple and easy-to-understand Python code
- Demonstrates basic data visualization techniques

## Technologies Used
- Python
- Matplotlib

## How to Run the Project
1. Install Python on your system.
2. Install Matplotlib using the command:
   pip install matplotlib
3. Download or clone the repository.
4. Run the Python file to generate the graphs.
